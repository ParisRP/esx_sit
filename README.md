Nouvelles fonctionnalités :

1. Support complet pour tous les props de votre liste
2. Configurations spécifiques par type de siège (chaise, bureau, canapé, etc.)
3. Scénarios GTA optimisés pour chaque type
4. Détection d'occupation - empêche deux joueurs de s'asseoir au même endroit
5. Système de distance - se lève automatiquement si trop loin
6. Options ox_target ajoutées dynamiquement pour chaque modèle
7. Debug mode pour le développement
8. Base de données optionnelle pour tracker les sessions
9. API exports pour d'autres scripts
10. Gestion des collisions améliorée

Commandes disponibles :

· /sit - S'asseoir sur le siège le plus proche
· Touche F2 par défaut (configurable)
